# /bc

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/bc your text here
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Sends a message to the EQBC server.
<!--cmd-desc-end-->

## Examples

- Sends the message "hello world" to the eqbc server
<code>/bc hello world</code>

# /bcg

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/bcg //command
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Send /command to all connected group members, excluding client issuing the command
<!--cmd-desc-end-->

## Examples

- Make everyone in the group except yourself sit.  
  ``/bcg //sit``

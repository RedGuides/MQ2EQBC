# /bca

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/bca //command
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Send /command to all connected clients, excluding client issuing the command
<!--cmd-desc-end-->

## Examples

- This makes all characters connected to the server say "hail" excluding the character that issued the command: `/bca //say hail`
